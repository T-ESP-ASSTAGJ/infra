# Ansible Lab Notes

The Ansible subtree is preserved here as a standalone lab environment rather than an integrated deployment path.

## Expected Local Inputs

- `vaulted_secrets.example.yml` documents the variables that were previously provided via an encrypted file
- `playbooks/rke2_join_info.example.yml` documents the worker join variables used by the RKE2 worker setup task
- `network_vars.yml` contains sample values generated by `scripts/network_setup.sh` and should be adapted per host

## Important Constraints

- inventory targets a Vagrant-based lab layout
- several manifests still contain placeholder endpoints or credentials and are not production-ready
- Vault initialization writes local key material during execution; those artifacts are ignored by `rnd/.gitignore`
